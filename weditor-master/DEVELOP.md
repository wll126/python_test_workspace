## Doc for Developers
```bash
# clone
git clone https://github.com/openatx/weditor
cd weditor

# install with -e
pip install -e weditor
```

`-e`这个选项可以将weditor目录下的代码直接关联到python的`site-packages`中。


修改完后，直接运行`python -m weditor`调试
